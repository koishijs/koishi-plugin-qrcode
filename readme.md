# koishi-plugin-qrcode

[![npm](https://img.shields.io/npm/v/koishi-plugin-qrcode?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-qrcode)

在 Koishi 中生成并发送二维码。

## 指令：qrcode

- 基本语法：`qrcode <text>`
- 选项列表：
  - -m, --margin \<margin>  边界尺寸
  - -s, --scale \<scale>  比例系数
  - -w, --width \<width>  图片大小
  - -d, --dark \<color>  暗部颜色
  - -l, --light \<color>  亮部颜色
